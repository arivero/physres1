# S30 Plan: Can a Fully Covariant Area-2-Form Postulate Work?

## Goal
Test whether a Lorentz-scalar postulate built directly from orbital 2-forms can replace source-frame projection in the areal-speed branch.

## Scope
1. Use orbital bivector \(M^{\mu\nu}=R^\mu p^\nu-R^\nu p^\mu\).
2. Evaluate Lorentz invariants \(I_1=\frac12 M_{\mu\nu}M^{\mu\nu}\), \(I_2=\frac12 M_{\mu\nu}{}^\star M^{\mu\nu}\).
3. Check whether these invariants retain velocity/areal-rate information for circular motion.

## Acceptance Tests
- Explicit statement whether \(I_1,I_2\) can encode the needed branch parameter.
- If not, identify minimal extra structure required.
