Derived the Schwarzschild Kepler law:
1. Orbital frequency: ω_φ = dφ/dt = (L/r₀²)/(E/(1-2M/r₀))
2. Using L² = Mr₀²/(r₀-3M), E² = (r₀-2M)²/(r₀(r₀-3M)):
   ω_φ² = (L²/r₀⁴) × (1-2M/r₀)²/E²
        = (M/(r₀²(r₀-3M))) × (r₀(r₀-3M)/(r₀-2M)²) × (1-2M/r₀)²/r₀²
        = M/r₀³
3. Proper period: dτ/dt = √(1-3M/r₀) on circular orbit, so T_proper = T_coord √(1-3M/r₀).
4. At ISCO: √(1-3M/6M) = √(1/2) = 1/√2 ≈ 0.707.
