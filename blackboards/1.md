## 1. Setup (Quantum-Mechanical Contact Interaction)

Hamiltonian:
\[
H = -\frac{\hbar^2}{2m}\Delta + g\,\delta^{(2)}(x) \quad \text{on } \mathbb{R}^2.
\]

**Issue:** Loop integrals diverge logarithmically → continuum theory ill-defined without renormalization.

## 2. Tree Structure (One-Loop = Single Rooted-Tree Node)

**Born series:** \(T\)-matrix expansion in powers of \(g\):
\[
T = T^{(0)} + T^{(1)} + T^{(2)} + \cdots
\]

**Order 0:** Free propagation (no vertices) — trivial tree (no nodes).

**Order 1 (one-loop):** Lippmann–Schwinger integral \(\int G_0 \, g \, G_0\).

**Rooted-tree diagram:**
\[
\tau_1 = \bullet \quad \text{(single vertex, one loop attached)}.
\]

**Key point:** At one loop, there are **no nested subdivergences** (no subtrees). The tree has only **one node**. This is the RG analogue of the order-1 Butcher tree \(\tau_1 = \bullet\).

## 3. Divergent Loop Integral

Cutoff evaluation (wavevector cutoff \(\Lambda\)):
\[
I(E;\Lambda) = \int_{|q|<\Lambda} \frac{d^2q}{(2\pi)^2} \, \frac{1}{E - \frac{\hbar^2 q^2}{2m} + i0}.
\]

Result:
\[
I(E;\Lambda) = -\frac{m}{2\pi\hbar^2} \left[ \ln\!\left(\frac{\Lambda^2}{k^2}\right) + i\pi \right] + O\!\left(\frac{k^2}{\Lambda^2}\right),
\]
where \(E = \hbar^2 k^2 / (2m)\).

**Divergence:** \(\sim -\frac{m}{2\pi\hbar^2} \ln\Lambda^2\) as \(\Lambda \to \infty\).

## 4. Counterterm (Single Subtraction)

\(T\)-matrix with bare coupling:
\[
T(E;\Lambda) = \frac{1}{g_B(\Lambda)^{-1} - I(E;\Lambda)}.
\]

Define **renormalized coupling** at scale \(\mu\):
\[
\frac{1}{g_R(\mu)} \equiv \frac{1}{g_B(\Lambda)} + \frac{m}{2\pi\hbar^2} \ln\!\left(\frac{\Lambda^2}{\mu^2}\right).
\]

Substituting cancels \(\ln\Lambda\):
\[
T(E) = \frac{1}{\frac{1}{g_R(\mu)} + \frac{m}{2\pi\hbar^2} \ln\!\left(\frac{\mu^2}{k^2}\right) + i\,\frac{m}{2\hbar^2}}.
\]

**Rooted-tree interpretation:**
- Bare coupling \(g_B\) corresponds to the "tree coefficient" \(a(\tau_1)\) before renormalization.
- Counterterm \(\delta(1/g) = (m/2\pi\hbar^2) \ln(\Lambda^2/\mu^2)\) is the **subtraction** at node \(\tau_1\).
- Renormalized amplitude \(T(E)\) is the "renormalized tree coefficient" (analogous to RK order condition).

## 5. Beta Function (RG Flow = Scale Composition)

**Scale-independence condition:** \(\mu \, d/d\mu\, T(E) = 0\).

Yields:
\[
\mu \frac{d}{d\mu} \left( \frac{1}{g_R(\mu)} \right) = -\frac{m}{\pi\hbar^2},
\]
hence
\[
\beta(g_R) \equiv \mu \frac{d g_R}{d\mu} = \frac{m}{\pi\hbar^2} \, g_R^2.
\]

**Tree interpretation:**
- Changing \(\mu_1 \to \mu_2\) is analogous to composing two RK steps \(\Phi_h \circ \Phi_h\).
- The beta function ensures **scale composition**: \(R_{\mu_2}(R_{\mu_1}^{-1}(\Gamma)) = R_{\mu_2}(\Gamma)\).
- At one loop (single tree \(\tau_1\)), the "composition law" is just the logarithmic running.

## 6. Dimensional Transmutation (RG-Invariant Scale)

For bound state \(E = -\hbar^2 \kappa^2 / (2m)\), \(T\) has a pole when:
\[
\frac{1}{g_R(\mu)} + \frac{m}{2\pi\hbar^2} \ln\!\left(\frac{\mu^2}{\kappa^2}\right) = 0.
\]

Define:
\[
\kappa_*^2 \equiv \mu^2 \exp\!\left( \frac{2\pi\hbar^2}{m} \frac{1}{g_R(\mu)} \right).
\]

**RG invariance:** \(d\kappa_* / d\mu = 0\).

**Physical content:** The model has **one physical scale** \(\kappa_*\) (equivalently, bound-state energy \(E_B = \hbar^2 \kappa_*^2 / (2m)\)). The bare coupling \(g_B\) is traded for this scale via dimensional transmutation.

**Rooted-tree analogy:**
- \(\kappa_*\) is the "physical prediction" (like the final \(y_{n+1}\) in RK).
- The subtraction scale \(\mu\) is arbitrary (like step size \(h\) in RK).
- Changing \(\mu\) rescales \(g_R(\mu)\) but leaves \(\kappa_*\) invariant (like changing \(h\) but keeping total integration time \(T\) fixed).

## 7. Scheme Dependence (Finite Counterterm Shift)

Shift the subtraction by a constant \(C\):
\[
\frac{1}{g_R^{(C)}(\mu)} \equiv \frac{1}{g_R(\mu)} + \frac{m}{2\pi\hbar^2} C.
\]

Beta function unchanged: \(\mu \, d/d\mu\, (1/g_R^{(C)}) = -m/(\pi\hbar^2)\).

RG-invariant scale rescales:
\[
\kappa_*^{(C)\,2} = e^C \, \kappa_*^2.
\]

**Rooted-tree analogy:**
- Scheme choice \(C\) is like choosing a different RK tableau with the same order (e.g., midpoint vs Heun for order-2).
- Both give the same "accuracy" (same beta function / same order condition), but differ by a finite shift (scheme / method).
- Fixing one physical datum (bound-state energy \(E_B\) / exact solution at a test point) removes the ambiguity.

## 8. Connection to Butcher Composition

**Order-1 RK (Euler method):**
\[
\Phi_h(y) = y + h\,f(y), \quad a(\tau_1) = 1.
\]

Composition:
\[
\Phi_h \circ \Phi_h = \Phi_{2h} + O(h^2).
\]

**One-loop RG (2D delta):**
\[
T(E;\mu) = \frac{1}{1/g_R(\mu) + (\text{log term})}, \quad \text{"coefficient"} = 1/g_R(\mu).
\]

Scale composition:
\[
R_{\mu_2}(R_{\mu_1}^{-1}(T)) = R_{\mu_2}(T) + O(g_R^2).
\]

**Parallel:**
- Both have **single-node tree** \(\tau_1 = \bullet\).
- Both satisfy **composition law** (Butcher: step doubling, RG: scale doubling).
- Both have **one free parameter** (step size \(h\) / subtraction scale \(\mu\)).
- Both trade regulator (\(h\) / \(\Lambda\)) for physical output (\(y_{n+1}\) / \(\kappa_*\)).
