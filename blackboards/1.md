# Rank-2 Instanton Lattice and Stokes Constants from Composition (C)

## 1. Setup

**Context:** D6.4b (Instanton Lattice Rigidity) proved: composition (C) forces
singularity positions {nζ_1} for rank-1 lattices (Borel/instanton type, simple-pole).
Two open questions from resurgence notebook §13:
(a) Rank-2: does (C) force {nS_1 + mS_2 : n,m ≥ 0} for two instanton actions?
(b) Stokes constants: does (C) constrain residues at nζ_1, not just positions?

**Notation:**
- K_t: propagator satisfying (C)
- B[K_t](ζ): Borel transform
- Δ_ω: alien derivative at ζ = ω
- S_n(t): Stokes constant — coefficient in Δ_{nA}(K_t) = S_n(t) · K_t
- ℤ_≥0^2: free abelian semigroup on {S_1, S_2}

## 2. Rank-2 Lattice: Main Claim

**Proposition (rank-2 corollary of D6.4b):**
If K_t has Borel singularities only at positions in {nS_1 + mS_2 : n,m ≥ 0} (rank-2 semigroup),
then (C) forces the singularity set to be closed under addition: it remains in {nS_1 + mS_2}.

**Proof:**
Borel convolution: B[K_{t₁+t₂}](ζ) = ∫_0^ζ B[K_{t₁}](ζ') B[K_{t₂}](ζ-ζ') dζ'

A singularity at ζ = ω arises from paired singularities at ζ' = ω_1 and ζ-ζ' = ω_2 = ω - ω_1.
If ω_1 = n_1 S_1 + m_1 S_2 and ω_2 = n_2 S_1 + m_2 S_2, then
ω = ω_1 + ω_2 = (n_1+n_2)S_1 + (m_1+m_2)S_2 ∈ {nS_1 + mS_2}. ✓

**Conclusion:** (C) is COMPATIBLE with rank-2 lattices and forces their additive closure.
This is a COROLLARY of D6.4b — the rank-2 case adds no new structural forcing.

**Critical observation:** The rank-2 structure says nothing about WHETHER two independent
instanton actions S_1, S_2 exist — that is physical input. (C) does not force rank(lattice) = 1,
2, or any specific rank. It forces CLOSURE given the generators.

## 3. Is Rank-2 Trivial?

**Summary:** YES — rank-2 (and rank-k for any k) is a direct corollary of the rank-1 argument.

D6.4b's proof relies on the additive semigroup property of Borel convolution. This property holds
for any set of generators {S_1, S_2, ..., S_k}. There is no new phenomenon at rank 2.

**What would be non-trivial:**
A constraint that FORCES the rank to be 1 (i.e., no two independent instanton actions can both
satisfy (C)). But this is FALSE: CP(1) sigma model has a single instanton action S = 4π/g²,
so rank-1; but CP(N) with multiple vacuum sectors can have rank-2 instanton lattices, and
all of these are compatible with (C). So (C) does not select the rank.

## 4. Stokes Constants: The Functional Equation

**Alien Leibniz Rule (from resurgence notebook §4):**
Δ_{nA}(K_{t₁+t₂}) = Σ_{j=0}^{n} Δ_{jA}(K_{t₁}) · Δ_{(n-j)A}(K_{t₂})

Writing Δ_{nA}(K_t) = S_n(t) · K_t (scalar Stokes constant at each sector):

S_n(t₁+t₂) = Σ_{j=0}^{n} S_j(t₁) · S_{n-j}(t₂)

**Generating function:** Define F(t, λ) = Σ_{n≥0} S_n(t) λ^n. Then:
F(t₁+t₂, λ) = F(t₁, λ) · F(t₂, λ)

**Cauchy functional equation:** F(t₁+t₂) = F(t₁)·F(t₂) for t₁,t₂ > 0.

Solutions (for measurable F):
- F(t, λ) = e^{c(λ)·t} for some function c(λ) ∈ ℂ

**Applying boundary condition F(0, λ) = 1 (identity K_0 = δ → no alien part):**
c(0) = 0 (the vacuum sector S_0 = constant). More precisely:
S_0(t) = e^{c_0 t} (can be set to 1 by normalization of K_t)
S_n(t) = [coefficient of λ^n in e^{c(λ)t}] — polynomial in t of degree ≤ n

**Example (rank-1 instanton, single c = c_1 λ):**
F(t, λ) = e^{c_1 λ t} = Σ_n (c_1 λ)^n t^n / n!
So S_n(t) = c_1^n t^n / n! — the Poisson distribution in t.

## 5. What (C) Forces on Stokes Constants

**FORCED by (C):**
1. TIME-DEPENDENCE: S_n(t) = [coefficient of λ^n in e^{c(λ)t}] — determined by c(λ)
2. POLYNOMIAL DEGREE: S_n(t) is a polynomial in t of degree ≤ n
3. CONVOLUTION STRUCTURE: S_n(t) = [n-th cumulant of] · e^{c(λ)t}
4. ZERO LIMIT: S_n(0) = δ_{n,0} (from identity K_0 = δ)

**NOT FORCED by (C):**
1. The function c(λ) itself — this is non-perturbative input
2. The VALUE of any Stokes constant S_n at any fixed t (requires c(λ))
3. Whether c(λ) has finitely or infinitely many poles in λ

**Key result:** The generating function F(t,λ) = e^{c(λ)t} makes the t-dependence of
all Stokes constants completely determined by the SINGLE function c(λ), which encodes the
non-perturbative content of the theory. This is a stronger statement than just "semigroup":
it says the Stokes sector organizes into a one-parameter exponential family in t.

### P8.1: Is the exponential-t family correct physically?

**Short answer: YES, with one important qualification.**

In physical models with a single instanton sector, the Stokes constants do satisfy
F(t, λ) = e^{c(λ)t} — but the function c(λ) is not a free complex function; it is
constrained by the INSTANTON ACTION and the COUPLING in a specific way.

**Physical Stokes constant in 2D delta (one-loop exact):**
The full K_t = e^{-tH} in Euclidean, with H = -Δ_δ (delta-interaction Hamiltonian).
The one-loop-exact result: K_t(x,y) = K_t^{free}(x,y) + e^{-|E_B|t} · φ_B(x)φ_B(y)†
where E_B = -ℏ²κ_*²/(2m) and φ_B is the bound state.

The "Stokes constant" at the instanton singularity ζ = E_B/ℏ is:
S_1(t) = residue ∝ e^{-|E_B|t/ℏ}

This is F(t) = e^{c·t} with c = -|E_B|/ℏ < 0 — exactly exponential in t, as predicted.
For higher sectors (n-instanton): S_n(t) = (c)^n t^n / n! (Poisson) — matches the
Borel-transform prediction that F(t,λ) = e^{cλt} with c = 1/g_R (identified with |E_B|/ℏ
via dimensional transmutation).

**Qualification:** The mathematician's claim F(t,λ) = e^{c(λ)t} is correct. But in
physically realizable models, c(λ) is NOT a general analytic function — it is:
- For 2D delta (one-loop exact): c(λ) = κ_* λ (linear in λ, one parameter)
- For double-well (multi-instanton): c(λ) = κ_* λ + c_2 λ² + ... (multi-instanton series)
- The OPE/S-matrix constrains c(λ) through the resurgence triangle (Dunne-Ünsal)

**Mathematician's open question** (§7: "Is c(λ) constrained by any additional structure?"):
YES, in models with exact crossing symmetry or Ward identities. But this is PHYSICAL
input, not structural forcing from (C). The mathematician is correct that (C) alone
does not constrain c(λ) beyond the exponential-t form.

### P8.2: Rank-2 clarification

The rank-2 claim (§2-3) is physically well-stated. A concrete rank-2 example:
- CP(1) sigma model in 2D: instantons S_1 = 4π/g² and anti-instantons S_2 = 4π/g² (same!)
  → actually rank-1 (S_1 = S_2 = S)
- Sine-Gordon model at generic coupling β²: instantons at S and "fractional" at S/2 (if
  certain rational β² values) — these give rank-2 if S and S/2 are incommensurable over ℤ.
  But S/2 ∈ ℤ·S is commensurable, so genuinely rank-2 requires SEPARATE instanton species.

**Honest assessment:** Genuinely rank-2 (incommensurable) lattices require models with two
DIFFERENT topological sectors (different vacuum winding numbers producing different action
scales). Standard QM models (double-well, delta interaction) are rank-1. CP(N) for N≥2 or
product theories CP(1)×CP(1) are rank-2 candidates. The mathematician's conclusion (rank-2
is a corollary, not a new result) is correct — the additive semigroup argument is identical.
