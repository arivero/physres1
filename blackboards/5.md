# Action-Angle Indeterminacy: Mathematical Verification (Pre-Vote)

## 1. [L_z, E] = ℏE (§2, key commutation relation)

E = e^{iφ}, L_z = -iℏ∂_φ, acting on 2π-periodic functions.

[L_z, E]ψ = L_z(Eψ) - E(L_zψ)
           = -iℏ∂_φ(e^{iφ}ψ) - e^{iφ}(-iℏ∂_φψ)
           = -iℏ[ie^{iφ}ψ + e^{iφ}∂_φψ] + iℏe^{iφ}∂_φψ
           = -iℏ·ie^{iφ}ψ = ℏEψ  ✓

**V1.1 VERIFIED:** [L_z, E] = ℏE. ✓

## 2. Circular Uncertainty Relation (Proposition 4.2)

**Claim:** Var(L_z)·(1-R²) ≥ ℏ²R²/4, where R = |⟨e^{iφ}⟩|.

**Commutators (§4, using in derivation):**
[L_z, cos φ]ψ = -iℏ∂_φ(cos φ ψ) + iℏcos φ ∂_φψ = iℏ sin φ ψ ✓
[L_z, sin φ]ψ = -iℏ∂_φ(sin φ ψ) + iℏsin φ ∂_φψ = -iℏ cos φ ψ ✓

**Robertson applied to (L_z, cos φ) and (L_z, sin φ):**
Var(L_z)·Var(cos φ) ≥ (ℏ²/4)⟨sin φ⟩²
Var(L_z)·Var(sin φ) ≥ (ℏ²/4)⟨cos φ⟩²

**Adding (Var(L_z) is common factor):**
Var(L_z)[Var(cos φ) + Var(sin φ)] ≥ (ℏ²/4)[⟨sin φ⟩² + ⟨cos φ⟩²]

**Identities:**
Var(cos φ)+Var(sin φ) = ⟨cos²φ+sin²φ⟩ - (⟨cosφ⟩²+⟨sinφ⟩²) = 1 - R²
⟨sinφ⟩²+⟨cosφ⟩² = |⟨e^{iφ}⟩|² = R²

Therefore: Var(L_z)·(1-R²) ≥ ℏ²R²/4. ✓

**V2.1 VERIFIED:** Proposition 4.2 derivation is correct. ✓

**V2.2 FLAG (sev-3):** The Robertson inequality requires that the commutator
expectation value ⟨[L_z, cos φ]⟩ = iℏ⟨sin φ⟩ is well-defined. For L_z eigenstates
(where the LHS is trivial, R=0), this is fine — the bound becomes 0≥0.
The Robertson inequality also requires the state to be in the domain of both L_z
and of cos φ, sin φ (which is automatic for square-integrable 2π-periodic functions).
No issue. Sev-3 precision note only.

## 3. Example 4.3: Dirichlet-Kernel Packet Verification

ψ_M(φ) = (1/√(2π(2M+1))) Σ_{m=-M}^{M} e^{imφ}

In terms of normalized Fourier coefficients: c_m = 1/√(2M+1) for m ∈ {-M,...,M}.

**R = |⟨e^{iφ}⟩|:**
⟨e^{iφ}⟩ = Σ_{m,m': |m|,|m'|≤M} c_m^* c_{m'} δ_{m'+1,m'...}

More directly: ⟨e^{iφ}⟩ = Σ_{|m|≤M} |c_m|² δ... no.

Using Fourier: ⟨e^{iφ}⟩ = Σ_{|m|≤M, |m+1|≤M} |c_m|² ← wrong framing.

Correct: ⟨e^{iφ}⟩ = Σ_{|m|≤M} Σ_{|n|≤M} c_m^* c_n ∫ e^{iφ} e^{i(n-m)φ} dφ/(2π)
= Σ_{|m|≤M, |n|≤M} c_m^* c_n δ_{n-m,1}
= Σ_{|m|≤M, |m+1|≤M} |c_m|²  [with c_m=1/√(2M+1)]
= (1/(2M+1)) · #{m: |m|≤M and |m+1|≤M}
= (1/(2M+1)) · 2M  [m ranges from -M to M-1, giving 2M values]

So |⟨e^{iφ}⟩| = 2M/(2M+1). ✓ (matches paper)

**Var(L_z):**
⟨L_z⟩ = ℏ/(2M+1) · Σ_{m=-M}^{M} m = 0 (symmetric). ✓
⟨L_z²⟩ = ℏ²/(2M+1) · Σ_{m=-M}^{M} m² = ℏ²·2·M(M+1)(2M+1)/(6(2M+1)) = ℏ²M(M+1)/3. ✓
Var(L_z) = ℏ²M(M+1)/3. ✓

**Ratio computation:**
1-R² = 1-(2M/(2M+1))² = (4M+1)/(2M+1)²

LHS = Var(L_z)(1-R²) = ℏ²M(M+1)(4M+1)/(3(2M+1)²)
RHS = ℏ²R²/4 = ℏ²M²/(2M+1)²

Ratio = (M+1)(4M+1)/(3M). ✓

At M=1: 2·5/3 = 10/3 ≈ 3.33. ✓
Large M: ≈ 4M/3 → ∞. ✓ (inequality satisfied with increasing slack)
