## Part A: Feynman Kernel Composition (c=i)

**Setup:** K(x,y;t) = N(t) exp(c(x-y)^2/t) with c = i (Minkowski)
N(t) = sqrt(-i/(pi*t))

**Composition identity:** N(t_1)*N(t_2)*J = N(t_1+t_2) where
J = integral_y exp(i(x-y)^2/t_1 + i(y-z)^2/t_2) dy = sqrt(pi*i*t_1*t_2/(t_1+t_2))

**Verification (SymPy ratio test):**
product / N_tot = -sqrt(I)*I / sqrt(-I)
|product/N_tot|^2 = 1  [SymPy: True]
Numerical check: |product - N_tot| < 1e-10 for (t1,t2) in {(1,2),(0.5,3)} [True]

**Algebraic identity:** -i*sqrt(i) = sqrt(-i) = e^{-i*pi/4}
Numerical: -i*sqrt(i) = 0.707107-0.707107j = sqrt(-i) ✓

**Conclusion:** Both c=-1 (heat kernel) and c=i (Feynman) satisfy composition.
Wick rotation (c=-1 <-> c=i) is A3 representation-channel, not forced by composition.

## Part B: Curvature Correction to Contact Amplitude Delta C_2

**Setup:** d=3 spatial, isotropic curvature R_{ij} = (R/3) delta_{ij}
Contact amplitude A(q) = C_0 + C_2 q^2 + ...
Physicist (blackboard 4) claims: Delta C_2 = C_2 * R * r_0^2 / 12

**Normal-coordinate Laplacian (corrected to O(R) via Christoffel + sqrt(g)):**
nabla^2_g - nabla^2_flat = (R/18)(|y|^2 nabla^2 - (y.nabla)^2) - (R/9) y.nabla + O(Ry^2, R^2)
[Standard result: -(1/3)R_{ij} y^j partial_i for Ricci R_{ij} = R/3 delta_{ij}]

**Fourier transforms for tilde_g = q^2 (C_2 term):**
- FT[(y.nabla) g] = d*q^2 + q.nabla_q(q^2) = 3q^2 + 2q^2 = 5q^2
- FT[(y.nabla)^2 g] = d*(5q^2) + q.nabla_q(5q^2) = 15q^2 + 10q^2 = 25q^2
- FT[|y|^2 nabla^2 g] = nabla^2_q(q^4) = 20q^2 [in 3D]

**Pure-delta correction:**
Delta C_2^{pure} = C_2 * [(R/18)*(20q^2-25q^2)/q^2 - (R/9)*5q^2/q^2]
= C_2 * [-(5R/18) - (5R/9)] = -5RC_2/6
[Dimensionless; absorbed into renormalization of C_2]

**Smeared contact (Gaussian, range r_0) via volume element:**
sqrt(g)(r) = 1 - (R/6)|r|^2 + O(R^2)
FT[r^2 exp(-r^2/r_0^2)] = pi^{3/2} r_0^5 (-q^2 r_0^2 + 6) exp(-q^2 r_0^2/4) / 4

Delta A_vol = -(R/6) * integral |r|^2 e^{iq.r} C_2(-nabla^2)delta_{r_0}(r) d^3r
= -(R/6) * (-C_2) * (r_0^2/2) at q=0  [from -nabla^2_q exp(-q^2 r_0^2/4)|_{q=0} = r_0^2/2]

=> **Delta C_2 = C_2 * R * r_0^2 / 12  ✓  [Physicist's formula verified]**

**Coefficient source:** (1/6) from sqrt(g) expansion * (1/2) from Gaussian FT Laplacian = 1/12.

**Distinction:**
- Pure contact: Delta C_2 = -5R/6 * C_2 (running, enters renormalization)
- Smeared contact: Delta C_2 = +R r_0^2/12 * C_2 (EFT range correction, r_0 = interaction range)
Both are physical; the r_0^2 formula applies when the contact has nonzero range.

**Special D=4 coincidence (physicist note, no correction needed):**
xi_HD = xi_conf = 1/6 in d=4. The curvature term hbar^2 R/6m is the conformal value.
Computationalist confirmation: the 1/6 factor is universal (same from volume element and from
ordering-prescription analysis). No numerical discrepancy found.
