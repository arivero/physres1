## 3. Theoretical Prediction

**Stahl's theorem (1997):** For Stieltjes functions with square-root branch point endpoint
(density ρ(s) ~ (s-1)^{1/2} near s=1), Padé [N/N] poles converge as ~ N^{-4/3}.

**Our fit:** N^{-1.56} (asymptotic estimate; finite-N effect pushes below -4/3).
**Best fit (Stieltjes):** Forced N^{-3/2} has RMS residual 0.068 (best forced fit).
**Forced N^{-4/3}:** RMS residual 0.167 (acceptable but worse than N^{-3/2}).

**Conclusion:** Convergence is approximately N^{-3/2}, consistent with N^{-4/3} at large N.

## 4. Minimum N for Threshold Detection

**Criterion:** |z_pole - 1| < 0.01 (1% accuracy):
- N^{-1.56} fit:  N ≈ 16
- N^{-3/2} fit:   N ≈ 20
- N^{-4/3} fit:   N ≈ 20

**Minimum N ~ 16-20** to detect threshold to 1% via Padé [N/N].

## 6. Implication for OQ4

**Confirmed (quantitative):** Padé [N/N] detects the threshold at z=1 but requires O(N^{4/3})
terms, not O(1). Exact non-perturbative recovery from perturbative Taylor coefficients requires
solving the full Stieltjes moment problem (N→∞ limit), not a finite Padé.

**This quantifies the structural distinction in OQ4:** pole = exactly recoverable from 2 terms;
branch cut = requires 16-20+ terms, exact only at N→∞.

### C8.2: Sev-3 — Coefficient Decay Rate (§1)

**Issue:** §1 states I_n ~ n^{-2} (algebraic decay).
B(n+1, 3/2) = Gamma(n+1)Gamma(3/2)/Gamma(n+5/2) ~ sqrt(pi)/2 · n^{-3/2} (Stirling).
Similarly B(n+2, 3/2) ~ n^{-3/2}. So I_n ~ n^{-3/2}, not n^{-2}.
This does not affect any downstream conclusion (Stahl theorem applies to both;
Pade convergence rate is determined by branch-point type, not coefficient decay).

## 9. Physicist Physical Verification
<!-- Physicist: 2026-02-20, Task #18 second-agent -->

**C8.2 confirmed (coefficient decay):**
B(n+1, 3/2) = Γ(n+1)Γ(3/2)/Γ(n+5/2). By Stirling: Γ(n+5/2)/Γ(n+1) ~ n^{3/2}.
So B(n+1,3/2) ~ √π/2 · n^{-3/2}. Similarly B(n+2,3/2) ~ n^{-3/2}.
Correct exponent: I_n ~ n^{-3/2}, not n^{-2}. Critic C8.2 confirmed.
This corresponds to branch-cut spectral weight ρ(s) ~ (s-1)^{1/2} (square-root endpoint,
standard for 2-particle threshold in QED vacuum polarization: Π(q²) has Im Π ~ (q²-4m²)^{1/2}).

**Physical verification of Stahl applicability:**
QED vacuum polarization: Π(q²) = (α/3π) ∫_{4m²}^∞ ds (1 + 2m²/s)(1 - 4m²/s)^{1/2} / (s - q²)
Near threshold s ≈ 4m²: integrand ~ (s-4m²)^{1/2} → Stieltjes function with square-root endpoint.
Stahl's theorem IS applicable. Exponent -4/3 is the correct asymptotic rate.
The observed -3/2 (finite-N) is consistent: for small N, subleading terms can steepen the apparent rate.

**Physical interpretation of N ~ 16-20:**
In practice, low-order perturbative QED is limited to N ≤ 4 (current state of multi-loop calculations).
This means Padé approximants from perturbation theory CANNOT reliably locate the branch cut within 1%.
The physical threshold is known analytically — Padé is useful as a cross-check, not as a primary method.
This supports the project's thesis: non-perturbative structure (the branch cut) is NOT recoverable from
finite perturbative order via Padé alone; it requires either the full resurgent trans-series (N→∞) or
analytic input (dispersion relations).
