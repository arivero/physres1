## 1. Setup

Position-dependent mass, 1D harmonic trap:
\[
H_0 = -\frac{\hbar^2}{2m}\partial_q^2 + \frac{1}{2}m\omega^2 q^2
\]

Mass function: \(f(q) = 1 + \alpha q^2\), so \(m(q) = m/f(q)\).

Full classical symbol: \(H = \frac{1}{2m}f(q)p^2 + \frac{1}{2}m\omega^2 q^2\).

Derivatives: \(f' = 2\alpha q\), \(f'' = 2\alpha\).

## 2. Weyl ordering (from blackboard 3, Remark D9.1a)

Weyl quantization of \(\frac{1}{2m}f(q)p^2\):
\[
\hat{H}_W = -\frac{\hbar^2}{2m}\left[f\,\partial_q^2 + f'\,\partial_q + \frac{f''}{4}\right]
= -\frac{\hbar^2}{2m}\left[(1+\alpha q^2)\partial_q^2 + 2\alpha q\,\partial_q + \frac{\alpha}{2}\right].
\]

**Verification (self-adjointness):** Operator \(a\partial^2 + b\partial + c\) is formally self-adjoint on \(L^2(\mathbb{R}, dq)\) iff \(b = a'\). Here \(a = f\), \(b = f' = a'\). Confirmed.

## 3. Half-density (Laplace-Beltrami conjugation)

### 3a. Laplace-Beltrami operator

Metric \(g = 1/f = (1+\alpha q^2)^{-1}\), inverse \(g^{-1} = f\). Determinant \(|g| = g\).

\[
\Delta_g = |g|^{-1/2}\partial_q\bigl(|g|^{1/2}g^{-1}\partial_q\bigr)
= g^{-1/2}\partial_q\bigl(g^{-1/2}\partial_q\bigr).
\]

Direct expansion:
\[
\Delta_g = f\,\partial_q^2 + \frac{f'}{2}\,\partial_q.
\]

**Derivation.** Set \(h = g^{-1/2} = f^{1/2}\). Then \(\Delta_g = h\,\partial_q(h\,\partial_q) = h^2\partial_q^2 + hh'\partial_q = f\,\partial_q^2 + \frac{f'}{2}\partial_q\).

### 3b. Conjugation to \(L^2(dq)\)

Map \(U\psi = g^{1/4}\psi = f^{-1/4}\psi\). Conjugated operator:
\[
\widetilde{H}_g = U\!\left(-\frac{\hbar^2}{2m}\Delta_g\right)\!U^{-1}
= -\frac{\hbar^2}{2m}\,f^{-1/4}\!\left(f\,\partial_q^2 + \frac{f'}{2}\partial_q\right)\!f^{1/4}.
\]

Set \(W = \tfrac{1}{4}\ln f\), so \(f^{1/4} = e^W\), \(W' = f'/(4f)\).

Conjugation identities:
\[
e^{-W}\partial_q\, e^W = \partial_q + W', \qquad
e^{-W}\partial_q^2\, e^W = \partial_q^2 + 2W'\partial_q + W'' + (W')^2.
\]

Result:
\[
\hat{H}_{HD} = -\frac{\hbar^2}{2m}\left[f\,\partial_q^2 + \underbrace{\left(2fW' + \frac{f'}{2}\right)}_{\text{first-derivative coeff}}\partial_q + f(W'' + W'^2) + \frac{f'}{2}W'\right].
\]

### 3c. First-derivative coefficient

\[
2fW' + \frac{f'}{2} = 2f\cdot\frac{f'}{4f} + \frac{f'}{2} = \frac{f'}{2} + \frac{f'}{2} = f'.
\]

**Both Weyl and half-density have the same first-derivative coefficient \(f'\).** Self-adjointness condition \(b = a'\) is satisfied by both.

### 3d. Scalar potential (Q_HD)

Collect zero-derivative terms:
\[
Q_{HD} = f(W'' + W'^2) + \frac{f'}{2}W'.
\]

Compute each piece:
- \(W' = \frac{f'}{4f}\), \quad W'' = \frac{f''}{4f} - \frac{f'^2}{4f^2}\)
- \(fW'' = \frac{f''}{4} - \frac{f'^2}{4f}\)
- \(fW'^2 = \frac{f'^2}{16f}\)
- \(\frac{f'}{2}W' = \frac{f'^2}{8f}\)

Sum:
\[
Q_{HD} = \frac{f''}{4} - \frac{f'^2}{4f} + \frac{f'^2}{16f} + \frac{f'^2}{8f}
= \frac{f''}{4} + \frac{f'^2}{f}\left(-\frac{1}{4} + \frac{1}{16} + \frac{1}{8}\right)
= \frac{f''}{4} - \frac{f'^2}{16f}.
\]

So:
\[
\hat{H}_{HD} = -\frac{\hbar^2}{2m}\left[f\,\partial_q^2 + f'\,\partial_q + \frac{f''}{4} - \frac{f'^2}{16f}\right].
\]

## 4. Operator difference

\[
\hat{H}_W - \hat{H}_{HD} = -\frac{\hbar^2}{2m}\left[\frac{f''}{4} - \left(\frac{f''}{4} - \frac{f'^2}{16f}\right)\right]
= -\frac{\hbar^2}{2m}\cdot\frac{f'^2}{16f}
= -\frac{\hbar^2 f'^2}{32mf}.
\]

**Purely scalar (multiplicative) operator.** No derivatives. Sign: \(H_W < H_{HD}\) pointwise.

For \(f = 1 + \alpha q^2\):
\[
\hat{H}_W - \hat{H}_{HD} = -\frac{\hbar^2}{32m}\cdot\frac{4\alpha^2 q^2}{1+\alpha q^2}
= -\frac{\hbar^2\alpha^2 q^2}{8m(1+\alpha q^2)}.
\]

### 4a. Order analysis

\(f'^2 = 4\alpha^2 q^2 = O(\alpha^2)\). The difference starts at \(O(\alpha^2)\), not \(O(\alpha)\).

**The first-order (O(\alpha)) energy shift vanishes identically.**

## 5. Ground-state expectation value

Unperturbed ground state: \(\psi_0(q) = (m\omega/\pi\hbar)^{1/4}\exp(-m\omega q^2/(2\hbar))\).

\[
\langle q^2\rangle_0 = \frac{\hbar}{2m\omega}, \qquad
\langle q^4\rangle_0 = \frac{3\hbar^2}{4m^2\omega^2}.
\]

To leading order in \(\alpha\) (expand \((1+\alpha q^2)^{-1} \approx 1 - \alpha q^2\)):
\[
\Delta E_0 = \left\langle 0\left|\hat{H}_W - \hat{H}_{HD}\right|0\right\rangle
= -\frac{\hbar^2\alpha^2}{8m}\langle q^2\rangle_0 + O(\alpha^3)
= -\frac{\hbar^2\alpha^2}{8m}\cdot\frac{\hbar}{2m\omega} + O(\alpha^3).
\]

\[
\boxed{\Delta E_0 = -\frac{\hbar^3\alpha^2}{16m^2\omega} + O(\alpha^3).}
\]

### Next order correction

Including \((1+\alpha q^2)^{-1} \approx 1 - \alpha q^2\):
\[
\Delta E_0 = -\frac{\hbar^2\alpha^2}{8m}\left[\langle q^2\rangle_0 - \alpha\langle q^4\rangle_0\right] + O(\alpha^4)
= -\frac{\hbar^2\alpha^2}{8m}\left[\frac{\hbar}{2m\omega} - \frac{3\alpha\hbar^2}{4m^2\omega^2}\right].
\]

## 6. Dimensionless form

Set \(\alpha = \alpha_0/a^2\) where \(a = \sqrt{\hbar/(m\omega)}\) is the oscillator length, \(\alpha_0\) dimensionless:
\[
\alpha^2 = \frac{\alpha_0^2 m^2\omega^2}{\hbar^2}.
\]

\[
\Delta E_0 = -\frac{\hbar^3}{16m^2\omega}\cdot\frac{\alpha_0^2 m^2\omega^2}{\hbar^2}
= -\frac{\hbar\omega\,\alpha_0^2}{16}.
\]

\[
\boxed{\Delta E_0 = -\frac{\hbar\omega\,\alpha_0^2}{16}.}
\]

Sign: \(E_0^{\text{Weyl}} < E_0^{\text{HD}}\). The Weyl ground state is lower.

## 8. Correction to blackboard 3

Blackboard 3 (Section II) states: "Key structural property: The first-derivative term d/dq is **eliminated**." This is **incorrect in 1D**.

**Corrected statement:** Both Weyl and half-density prescriptions yield the SAME first-derivative term \(f'\partial_q\). The half-density conjugation produces a self-adjoint operator (because \(b = a' = f'\)), but it does NOT eliminate the first-derivative. The prescriptions differ only in the O(\(\hbar^2\)) scalar potential:
\[
V_W^{(\hbar^2)} = -\frac{\hbar^2}{2m}\frac{f''}{4}, \qquad
V_{HD}^{(\hbar^2)} = -\frac{\hbar^2}{2m}\left(\frac{f''}{4} - \frac{f'^2}{16f}\right).
\]

Blackboard 3's four-layer stratification remains valid with this correction:
- Layer 1 (principal symbol): identical
- Layer 2 (connection/first-derivative): **identical** (both \(= f'\partial_q\))
- Layer 3 (scalar O(\(\hbar^2\))): **differ by** \(\hbar^2 f'^2/(32mf)\)
- Layer 4 (domain): independent

## 9. Verdict

**Verdict 4 + 1.** First-order shift (\(O(\alpha)\)) vanishes identically: \(\Delta E_0^{(1)} = 0\). Leading nonzero contribution is \(O(\alpha^2)\), with:
\[
|\Delta E_0| = \frac{\hbar\omega\alpha_0^2}{16} \approx 0.006\text{--}0.06\text{ meV} \ll 0.1\text{ meV}
\]
in the perturbative regime (\(\alpha_0 \lesssim 0.3\)).

D35's layer-3 deformation freedom has **no measurable consequence** for realistic position-dependent-mass systems within perturbation theory. Deformation equivalence is physical, not merely formal.
