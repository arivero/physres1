# Identity Limit from Composition + Positivity (No Stone's Theorem)

Date: 2026-02-20
Task: #23 (self-directed)
Question: can K_t → δ as t→0 be derived from (C) + positivity, bypassing Stone's theorem?

---

## 1. The Three Routes to Identity Limit

For action-based kernels satisfying composition (C), three routes exist:

**Route A (current, free particle):** Explicit Gaussian form → nascent delta. No additional axiom.
**Route B (current, general V):** Stone's theorem for self-adjoint H → U(t) strongly continuous.
  Requires: self-adjointness of H (not just symmetry). This is the gap in blackboard 0 / RCP-R6.2.
**Route C (proposed):** Composition + positivity (or normalization) → C₀-semigroup → identity limit.
  Question: does this avoid Stone? And is positivity itself an axiom or a theorem?

---

## 2. Hille–Yosida and C₀-Semigroups

**Definition.** A family {T(t)}_{t≥0} of bounded linear operators on a Banach space X is a
C₀-semigroup (strongly continuous semigroup) if:
- T(t₁+t₂) = T(t₁)T(t₂) for all t₁,t₂ ≥ 0
- T(0) = I (identity)
- t ↦ T(t)x is continuous at t=0 for each x ∈ X: lim_{t→0⁺} ‖T(t)x - x‖ = 0

**Key observation:** The identity limit T(0) = I is BUILT INTO the definition. But T(0) = I is
exactly the statement K_t → δ as t → 0 (in the weak sense needed for operator continuity).

**So:** "Is (I) forced by (C)?" is equivalent to: "Does the composition semigroup {U(t)}_{t>0}
extend to a C₀-semigroup at t=0?" This is asking whether U(0) := lim_{t→0+} U(t) exists and
equals I — which is exactly hypothesis (I).

**Hille-Yosida does NOT help directly:** Hille-Yosida characterizes which generators A yield
C₀-semigroups T(t) = e^{tA}. But it assumes the semigroup is already C₀ — it does not derive
C₀ from composition alone. The C₀ property at t=0 is an input, not an output.

## 3. Can Positivity Replace Stone?

**Idea:** For positive kernels (Markov/Euclidean context), there is a positivity-based route.

### 3.1 Positive kernel version (Markov semigroup theory)

A kernel K_t(x,y) ≥ 0 with ∫K_t(x,y)d^d y = 1 (stochastic/Markov) satisfying composition
defines a Markov semigroup on L∞(ℝ^d) or C₀(ℝ^d).

**Theorem (Feller semigroup):** If K_t is a Markov kernel satisfying:
- (C) K_{t₁+t₂} = K_{t₁} * K_{t₂} (convolution/composition)
- (P) K_t(x,y) ≥ 0, ∫K_t(x,y)d^d y = 1 for all x,t
- (Cont) K_t f → f pointwise as t→0 for all f ∈ C₀(ℝ^d) (Feller continuity)

Then {T_t}_{t≥0} defined by (T_t f)(x) = ∫K_t(x,y)f(y)d^d y is a C₀-semigroup on C₀(ℝ^d).

**But:** The "Feller continuity" condition (T_t f → f pointwise) is equivalent to the identity
limit K_t → δ. So (Cont) = (I). Positivity alone does NOT force (I) without (Cont).

### 3.2 The Wiener measure construction

For Brownian motion: K_t = heat kernel = Gaussian (positive, normalized). Identity limit
follows from the explicit Gaussian form (Route A). Positivity was not the key ingredient —
the explicit form was.

### 3.3 Chernoff's theorem route

Chernoff (1974): if F: [0,∞) → B(X) satisfies:
- ‖F(t)‖ ≤ e^{ωt}
- F(0) = I

Then the product formula F(t/n)^n converges to a C₀-semigroup T(t) as n→∞.

**Again:** F(0) = I is assumed, not derived. Chernoff's theorem gives a C₀-semigroup from an
approximating sequence but requires the identity at t=0 as an input.

## 4. The Fundamental Obstruction

**Claim:** No theorem in abstract semigroup theory derives the identity limit (C₀ property at
t=0) from the semigroup law alone, without additional conditions equivalent to (I).

**Reason:** The composition law T(t₁+t₂) = T(t₁)T(t₂) for t₁,t₂ > 0 says nothing about
the t→0 limit. There exist semigroups satisfying the composition law but failing C₀ at 0:

**Counterexample.** On X = L²(ℝ), define T(t) = multiplication by e^{t/|t|} for t > 0
(i.e., T(t) = e·I). Then T(t₁)T(t₂) = e²·I ≠ e·I = T(t₁+t₂) — this fails, but
one can construct more subtle examples with:

Correct counterexample: On X = L∞(ℝ), define T(t)f(x) = f(x+t) (shift semigroup). This is
C₀ on C₀(ℝ) but NOT on L∞(ℝ). The shift semigroup satisfies (C) but lim_{t→0}T(t)f in
L∞-norm does not exist for general f ∈ L∞.

**What this shows:** The composition law (C) does not determine the behavior at t=0. One can
have (C) satisfied for all t>0 without any continuous extension to t=0.

## 5. What DOES Force the Identity Limit?

The identity limit is forced by ONE of the following (equivalent for action-based kernels):

**(I₁) Stone's theorem route:** H self-adjoint → U(t) = e^{-iHt} strongly continuous.
  Identity limit: ‖U(t)φ - φ‖ = ‖(e^{-iHt}-1)φ̂‖₂ → 0 by DCT. Uses (R3).

**(I₂) Explicit kernel form:** For free particle, K_0 ∝ exp(im|x-y|²/(2ℏt)) is a
  nascent delta family. Follows from: semigroup law forces Gaussian form (D29/S288),
  and Gaussians are nascent deltas. Requires: semigroup forces the specific kernel form first.

**(I₃) Short-time asymptotics of the action:** As t→0, the classical action
  S_cl(x,y;t) = m|x-y|²/(2t) → +∞ for x≠y and → 0 for x=y (saddle-point localization).
  Combined with the exponential weight K ∝ exp(iS/ℏ), the kernel concentrates near x=y.
  Precise statement: stationary phase as t→0 gives K_t → δ for smooth test functions.

**Route (I₃) is the cleanest physical route:** It uses:
- Exponential weight (theorem of (C) via D29/S288)
- Short-time behavior of classical action S_cl(x,y;t) as t→0
- Stationary phase / Riemann-Lebesgue lemma

This route does NOT require Stone's theorem. It requires knowing the classical action S_cl
blows up as t→0 for x≠y — which is a property of the physical setup (Lagrangian mechanics),
not of the abstract semigroup.

## 6. Precise Route (I₃): Stationary Phase as t→0

**Hypothesis:** K(x,y;t) = N(t) exp(iS_cl(x,y;t)/ℏ) where S_cl(x,y;t) is the classical
  action of a non-degenerate Lagrangian L = ½m|ẋ|² - V(x) with V bounded below.

**Claim:** As t → 0⁺, ∫K(x,y;t)φ(y)d^d y → φ(x) for φ ∈ C_c(ℝ^d).

**Proof sketch:**
Step 1: N(t) ∝ t^{-d/2} (forced by composition, D29).
Step 2: Short-time asymptotics of S_cl(x,y;t) (standard result):
        S_cl(x,y;t) = m|x-y|²/(2t) + O(t) as t→0 (for bounded V, diagonal term dominates).
Step 3: Substitution u = (y-x)/√t gives:
        ∫K(x,y;t)φ(y)d^d y = ∫(m/2πiℏ)^{d/2} exp(im|u|²/(2ℏ)) φ(x+u√t) d^d u (leading term)
Step 4: As t→0, φ(x+u√t) → φ(x) uniformly on compacta. Dominated convergence
        (the Gaussian gives integrable domination for Re(iℏ) < 0, i.e. ε-prescription).
Step 5: ∫(m/2πiℏ)^{d/2} exp(im|u|²/(2ℏ)) d^d u = 1 (Gaussian integral with iε).

**Conclusion:** ∫K(x,y;t)φ(y)d^d y → φ(x). Identity limit proved from:
- Exponential form (theorem of (C))
- Physical setup: S_cl = m|x-y|²/(2t) + O(t) (Lagrangian mechanics with mass m)
- iε prescription (theorem of (C) via degree-counting, D37/S292)

No Stone's theorem invoked. No self-adjointness needed.

## 7. Verdict

**Route (I₃) closes the gap for action-based kernels without Stone's theorem:**

| Gap from RCP-R6.2 | Status with Route (I₃) |
|--------------------|------------------------|
| (R3) Self-adjointness | NOT NEEDED for identity limit if kernel is action-based |
| Nascent delta (free V) | Route (I₂): direct, clean |
| Stone's theorem (general V) | Route (I₃): replaced by stationary phase + Lagrangian asymptotics |

**Remaining condition for Route (I₃):**
- Physical setup must provide S_cl(x,y;t) = m|x-y|²/(2t) + O(t) as t→0 (short-time expansion)
- Valid for V bounded below (Kato class), which covers all standard potentials
- Fails for distributional V (singular potentials like delta) — those need Stone (R3)

**Refined regularity table (update to RCP-R6.2):**

For smooth/bounded V:
- Identity limit follows from: (C) + physical setup (exponential form + Lagrangian action asymptotics)
- Stone's theorem NOT needed
- Self-adjointness (R3) NOT needed

For singular V (delta, inverse-square, etc.):
- Short-time asymptotics of S_cl break down (classical paths may not exist)
- Stone's theorem route is the correct one
- Self-adjointness (R3) remains needed

## 8. Consequence for Single-Axiom Claim

**Strengthened claim (for smooth V):**
"Composition (C) + action-based physical setup with smooth V → Feynman kernel uniquely,
including identity limit, via: exponential form (D29) + stationary phase (Route I₃)."

Self-adjointness (R3) is NOT needed for smooth V. The physical setup provides the identity
limit through the short-time action asymptotics.

**Remaining gap (singular V):** For singular potentials (point interactions, self-adjoint
extension physics), Route (I₃) breaks down and Stone's theorem is genuinely needed. This is
not a defect of the theory — singular potentials are genuinely harder and self-adjoint
extension selection IS the physics of the boundary condition.

**Recommended manuscript addition:**
New Remark RCP-R6.3: "For smooth V, the identity limit follows from stationary phase as t→0
(Route I₃), bypassing Stone's theorem. For singular V requiring self-adjoint extension
selection, Stone's theorem (condition R3 of Remark RCP-R6.2) is genuinely needed."
