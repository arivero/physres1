## 1. Setup

Levy stable semigroup kernel in Fourier space:
\[
\hat{K}_t(p) = \exp\bigl(t\,\Psi(p)\bigr), \qquad \Psi(p) = -c\,|p|^\alpha, \quad \alpha \in (0,2].
\]

Semigroup property (C): \(\hat{K}_{t_1+t_2}(p) = \hat{K}_{t_1}(p)\,\hat{K}_{t_2}(p)\). Automatic.

Fundamental dimensions: \([p] = MLT^{-1}\) (momentum), \([t] = T\) (time), \([m] = M\) (mass), \([\hbar] = ML^2T^{-1}\) (action).

## 2. General Levy Scale: Dimensional Analysis

### 2a. Dimension of \(\Psi(p)\)

Exponent \(t\,\Psi(p)\) is dimensionless. Therefore:
\[
[\Psi(p)] = T^{-1}.
\]

### 2b. Dimension of \(c\)

From \([c\,|p|^\alpha] = T^{-1}\):
\[
[c] = T^{-1}\cdot[p]^{-\alpha} = T^{-1}\cdot(MLT^{-1})^{-\alpha} = M^{-\alpha}\,L^{-\alpha}\,T^{\alpha-1}.
\]

### 2c. Matching to fundamental constants \(m, \hbar\)

Hypothesis (D) of P4.2: the kernel parameters are built from available physical constants \(m\) (mass) and \(\hbar\) (action). Seek \(a, b\) such that:
\[
[c] = [m]^a\,[\hbar]^b = M^{a+b}\,L^{2b}\,T^{-b}.
\]

Matching exponents:
\[
\begin{cases}
M: & a + b = -\alpha \\
L: & 2b = -\alpha \implies b = -\alpha/2 \\
T: & -b = \alpha - 1 \implies b = 1 - \alpha
\end{cases}
\]

Consistency of L and T equations:
\[
-\alpha/2 = 1 - \alpha \quad\Longrightarrow\quad \alpha/2 = 1 \quad\Longrightarrow\quad \boxed{\alpha = 2}.
\]

### 2d. Solution for \(\alpha = 2\)

\(b = -1\), \(a = -2 - (-1) = -1\). So \(c \propto m^{-1}\hbar^{-1}\), i.e.:
\[
c = \frac{1}{2m\hbar}, \qquad \Psi(p) = -\frac{p^2}{2m\hbar}.
\]

Verification: \([1/(m\hbar)] = M^{-1}\cdot(ML^2T^{-1})^{-1} = M^{-2}L^{-2}T = [c]\big|_{\alpha=2}\). \(\checkmark\)

This is the free-particle Feynman-Kac/Schrodinger propagator (up to factor \(i\)).

### 2e. Obstruction for \(\alpha \neq 2\)

For \(\alpha \neq 2\), the system \(\{b = -\alpha/2,\; b = 1-\alpha\}\) has no solution. The Levy scale \(c\) cannot be expressed as any power of \(m\) and \(\hbar\).

**The dimensional obstruction is categorical:** no additional dimensionless parameters can remedy it, because the inconsistency is between the L-equation and the T-equation for the single unknown \(b\).

## 3. Specific Cases

### 3a. Cauchy kernel (\(\alpha = 1\))

\[
\hat{K}_t(p) = \exp(-c\,t\,|p|), \qquad [c] = M^{-1}L^{-1}T^{0} = (ML)^{-1}.
\]

From L: \(b = -1/2\). From T: \(b = 0\). Inconsistent.

Position space (Fourier inversion, \(d=1\)):
\[
K_t(x,y) = \frac{1}{\pi}\,\frac{ct}{c^2t^2 + (x-y)^2}.
\]

For this to be a Lorentzian with \([t]\) and \([x-y]\) in the same denominator: \([ct] = [L]\), hence \([c] = LT^{-1}\) (velocity). But from the Fourier exponent: \([c] = (ML)^{-1}\). The two are compatible only if \(M = T/L^2\), i.e., \([M] = T/L^2\), which is \([1/\hbar]\). This is circular: it assumes \(\hbar = 1\).

**In physical dimensions, the Cauchy kernel mixes position and time dimensions incommensurably unless \(\hbar\) and \(m\) are already fixed to specific numerical values.** This is a manifestation of the \(\alpha = 1\) obstruction.

### 3b. Stable law (\(\alpha = 3/2\))

\[
[c] = M^{-3/2}L^{-3/2}T^{1/2}.
\]

From L: \(b = -3/4\). From T: \(b = -1/2\). Inconsistent.

No combination of \(m\) and \(\hbar\) produces \([c]\).

### 3c. Summary table

| \(\alpha\) | \([c]\) | \(b\) from L | \(b\) from T | Consistent? |
|---|---|---|---|---|
| 1/2 | \(M^{-1/2}L^{-1/2}T^{-1/2}\) | \(-1/4\) | \(1/2\) | NO |
| 1 | \(M^{-1}L^{-1}\) | \(-1/2\) | \(0\) | NO |
| 3/2 | \(M^{-3/2}L^{-3/2}T^{1/2}\) | \(-3/4\) | \(-1/2\) | NO |
| **2** | \(M^{-2}L^{-2}T\) | \(-1\) | \(-1\) | **YES** |

## 4. Identity Limit Analysis

### 4a. Poisson jump process (\(d=1\))

Kernel:
\[
K_t(x,y) = e^{-\lambda t}\sum_{n=0}^{\infty}\frac{(\lambda t)^n}{n!}\,P_n(x,y),
\]
where \(P_0(x,y) = \delta(y-x)\), \(P_1(x,y) = \delta(y-x-a)\) (single-site jump, displacement \(a\)).

Short-time expansion:
\[
K_t(x,y) = e^{-\lambda t}\bigl[\delta(y-x) + \lambda t\,\delta(y-x-a) + O(t^2)\bigr]
= \delta(y-x) + \lambda t\bigl[\delta(y-x-a) - \delta(y-x)\bigr] + O(t^2).
\]

Weak identity limit: for \(\varphi \in C_c^\infty\),
\[
\lim_{t\to 0^+}\int K_t(x,y)\,\varphi(y)\,dy
= \lim_{t\to 0^+}\bigl[\varphi(x) + \lambda t\,(\varphi(x+a)-\varphi(x)) + O(t^2)\bigr]
= \varphi(x).
\]

**The Poisson kernel satisfies the identity limit (I).** The jump term is \(O(t)\) and vanishes as \(t \to 0^+\).

### 4b. Cauchy kernel (\(\alpha = 1\), \(d=1\))

Position-space kernel:
\[
K_t(x,y) = \frac{1}{\pi}\,\frac{t}{t^2 + (x-y)^2}.
\]

Normalization: \(\int_{-\infty}^{\infty}K_t(x,y)\,dy = 1\) for all \(t > 0\).

Weak identity limit: for \(\varphi \in C_c^\infty\),
\[
\int K_t(x,y)\,\varphi(y)\,dy = \frac{1}{\pi}\int\frac{t}{t^2+u^2}\,\varphi(x+u)\,du.
\]

Substitute \(v = u/t\):
\[
= \frac{1}{\pi}\int\frac{1}{1+v^2}\,\varphi(x+tv)\,dv
\xrightarrow{t\to 0^+} \frac{1}{\pi}\int\frac{1}{1+v^2}\,dv\;\varphi(x) = \varphi(x).
\]

**The Cauchy kernel satisfies the identity limit (I).** The Lorentzian is a standard nascent delta family.

### 4c. General \(\alpha\)-stable kernel

For any \(\alpha \in (0,2]\), the kernel \(\hat{K}_t(p) = \exp(-c\,t\,|p|^\alpha)\) with \(c > 0\) satisfies:
\[
\hat{K}_t(p) \xrightarrow{t\to 0^+} 1 \quad \text{for all } p.
\]

By the continuity theorem for characteristic functions, this implies \(K_t \to \delta\) in the weak-\(*\) sense.

**All \(\alpha\)-stable semigroups satisfy the identity limit (I).** The identity limit does not distinguish \(\alpha = 2\) from \(\alpha \neq 2\).

## 5. Dimensional Obstruction: Detailed Mechanism

### 5a. Gaussian case (\(\alpha = 2\)): dimensional chain

\[
\hat{K}_t(p) = \exp\!\left(-\frac{p^2\,t}{2m\hbar}\right)
\quad\Longleftrightarrow\quad
K_t(x,y) = \left(\frac{m}{2\pi\hbar t}\right)^{d/2}\exp\!\left(-\frac{m(x-y)^2}{2\hbar t}\right).
\]

Dimension check:
- Phase: \([p^2 t/(m\hbar)] = (MLT^{-1})^2\cdot T/(M\cdot ML^2T^{-1}) = 1\). \(\checkmark\)
- Prefactor: \([(m/(\hbar t))^{d/2}] = [L^{-d}]\). \(\checkmark\)

### 5b. Why \(\alpha \neq 2\) fails

For \(\alpha \neq 2\), the position-space kernel has the scaling:
\[
K_t(x,y) = t^{-d/\alpha}\,\mathcal{L}_\alpha\!\left(\frac{|x-y|}{(ct)^{1/\alpha}}\right),
\]
where \(\mathcal{L}_\alpha\) is the stable density (known in closed form only for \(\alpha = 1, 2\)).

Prefactor dimension:
\[
[K_t] = [t^{-d/\alpha}] = T^{-d/\alpha}.
\]

Requirement: \([K_t] = L^{-d}\). So \(T^{-d/\alpha} = L^{-d}\), giving \(T^{1/\alpha} = L\).

For \(\alpha = 2\): \(T^{1/2} = L\), which is satisfied by \(\sqrt{\hbar t/m}\) (the quantum spreading length).

For \(\alpha \neq 2\): \(T^{1/\alpha} = L\) requires a constant with dimension \([LT^{-1/\alpha}]\). No such constant exists among \(\{m, \hbar\}\):
- \([\hbar/m] = L^2T^{-1}\) gives \(L = T^{1/2}\) (i.e., \(\alpha = 2\))
- \([\hbar^a m^b]\) gives \(L^{2a}T^{-a} M^{a+b}\), so \(L/T^{1/\alpha} = L^{2a}T^{-a}/T^{1/\alpha}\) requires \(2a=1\) and \(a = 1/\alpha\), hence \(\alpha = 2\).

### 5c. Physical interpretation

The quantum mechanical length scale \(\ell(t) = \sqrt{\hbar t/m}\) (Gaussian spreading) is the unique length built from \(\{m, \hbar, t\}\) with the property that \(\ell \propto t^{1/2}\). Levy processes with \(\alpha \neq 2\) require spreading \(\ell \propto t^{1/\alpha}\), which demands a qualitatively different dimensional constant.

## 6. Connection to Blackboard 1 (S288) Loophole

Blackboard 1, Section 10 identified the Levy-Khintchine loophole: dropping hypothesis (L\_loc) admits the full Levy-Khintchine class
\[
\hat{K}_t(p) = \exp\!\left(t\left[i\mu p - \tfrac{\sigma^2}{2}p^2 + \int(e^{ipx}-1-ipx\,\mathbf{1}_{|x|<1})\,\nu(dx)\right]\right).
\]

**This blackboard closes the loophole:** hypothesis (D) (dimensional homogeneity) forces \(\alpha = 2\), eliminating the Levy measure \(\nu\) and recovering the Gaussian/Feynman-Kac propagator. The drift term \(i\mu p\) is eliminated by spatial symmetry or parity.

Therefore (L\_loc) is **derivable from (D):** any kernel satisfying (C)+(I)+(D) must have \(\alpha = 2\) (Gaussian characteristic exponent), which is precisely the class of action-dependent kernels.
