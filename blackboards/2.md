## 2. Isomonodromy — Core Definitions

**Def 2.1.** A linear ODE system: ∂_x Ψ = A(x,t) Ψ, A(x,t) rational in x.

**Def 2.2.** Isomonodromy: ∂_t A = ∂_x B + [A,B] (zero-curvature / Lax condition). Monodromy matrices M_j ∈ GL(n,ℂ) preserved under t-deformation.

**Def 2.3.** Painlevé I–VI = reductions of isomonodromy conditions for 2×2 systems on ℙ¹.

**Source:** Jimbo-Miwa-Ueno (1981), Harnad (1993), Bertola-Tovbis arXiv:1506.06897

## 3. Composition Compatibility

**P3.1 (Lax → semigroup).** Zero-curvature condition ∂_t A - ∂_x B + [A,B] = 0 implies:
- Parallel transport U(x₂,x₁;t) = P exp(∫_{x₁}^{x₂} A dx) satisfies composition:
  U(x₃,x₁;t) = U(x₃,x₂;t) · U(x₂,x₁;t)

**P3.2 (Monodromy groupoid).** Stokes data form a GROUPOID: objects = singular points {a_j}, morphisms = Stokes matrices S_j ∈ GL(n,ℂ), composition = matrix multiplication along paths.

**P3.3 (Composition law).** The monodromy representation π₁(ℙ¹ \ {sing. pts.}) → GL(n,ℂ) IS a groupoid homomorphism. Parallel transport = integration of the Lax connection.

**VERDICT on (C):** Isomonodromy systems ARE compatible with RCP composition (A1). Parallel transport satisfies (C) exactly.

## 4. RCP Channel Assessment

**Partition channel (A1):** COMPATIBLE. U(x₃,x₁) = U(x₃,x₂)∘U(x₂,x₁). Kernel = Green's function of linear system.

**Representation channel (A3):** COMPATIBLE. Different gauges (conjugation by GL(n)) give equivalent monodromy data. Representation-channel maps = gauge transformations.

**Scale channel (A4):** STATUS UNCLEAR.
- Isomonodromy is PARAMETER INDEPENDENT (monodromy preserved). This is OPPOSITE to RG flow (scale-dependent parameters).
- Birkhoff factorization (formal monodromy) ↔ RG-type equations? Candidate: Frenkel-Ben-Zvi formal monodromy.
- Key question: is isomonodromy a FIXED POINT of scale channel (β=0) or ORTHOGONAL to it?

## 5. Does (C) Force Isomonodromy?

**Negative result (single-agent):**

**P5.1.** (C) alone does NOT force isomonodromy.
- (C) is satisfied by ALL propagators (heat kernels, Schrödinger, generic path integrals).
- Isomonodromy requires: (i) auxiliary spectral parameter x, (ii) Lax structure, (iii) specific rational A(x,t).
- These are ADDITIONAL structure beyond (C).

**P5.2.** Additional axiom needed: **Integrability axiom (I_int)**: system admits Lax pair (∂_t - ∂_x + [A,B] = 0).
- (C) + (I_int) → Painlevé isomonodromy (in 2×2 case with rational connections on ℙ¹)
- (I_int) is NOT derivable from (C) alone; it is an independent structural assumption.
