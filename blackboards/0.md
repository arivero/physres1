# Dimensional Obstruction to Discrete-Time Composition
Date: 2026-02-14
Cycle: S296 (parent: D40)

## 1. Setup

P4.2 hypothesis (D): kernel parameters built from \(\{m, \hbar\}\) only, with \([K] = L^{-d}\).

Semigroup (C): \(K(x,z;T) = \int K(x,y;t_1)\,K(y,z;t_2)\,dy\), \(T = t_1 + t_2\), for all \(t_1, t_2 > 0\).

Identity (I): \(\lim_{t \to 0^+} K_t = \delta\).

**Question:** Can a discrete-time formulation introduce lattice spacing \(\varepsilon\) (time step) while satisfying (D)?

## 2. Dimensional Analysis of \(\varepsilon\)

### 2a. Available constants

| Constant | Dimensions |
|---|---|
| \(m\) | \(M\) |
| \(\hbar\) | \(ML^2T^{-1}\) |

Target: \([\varepsilon] = T\).

### 2b. Matching equations

Seek \(a, b \in \mathbb{R}\) such that \([\varepsilon] = [m]^a\,[\hbar]^b\):
\[
[m]^a\,[\hbar]^b = M^a \cdot (ML^2T^{-1})^b = M^{a+b}\,L^{2b}\,T^{-b}.
\]

Equating to \([\varepsilon] = M^0\,L^0\,T^1\):
\[
\begin{cases}
M: & a + b = 0 \\
L: & 2b = 0 \\
T: & -b = 1
\end{cases}
\]

### 2c. Inconsistency

From \(L\): \(b = 0\).
From \(T\): \(b = -1\).

\[
\boxed{b = 0 \quad\text{and}\quad b = -1 \quad\Longrightarrow\quad \text{contradiction.}}
\]

**No solution exists.** The time dimension \(T\) cannot be constructed from \(\{m, \hbar\}\).

### 2d. Root cause

\(\{m, \hbar\}\) span a 2-dimensional subspace of the 3-dimensional MLT system:
\[
\mathrm{span}\{[m], [\hbar]\} = \mathrm{span}\{M,\; ML^2T^{-1}\} \subset \{M, L, T\}.
\]

The image of the map \((a,b) \mapsto M^{a+b}L^{2b}T^{-b}\) is:
\[
\{M^\alpha\,L^\beta\,T^\gamma : \beta = -2\gamma\}.
\]

Since \([\varepsilon] = T = M^0 L^0 T^1\) requires \(\beta = 0\) and \(\gamma = 1\), the constraint \(\beta = -2\gamma = -2 \neq 0\) is violated. The dimension \(T\) is not in the image.

### 2e. Dimensional reach of \(\{m, \hbar\}\)

The only dimensionful quantities constructible from \(\{m, \hbar\}\) satisfy \([L] = [T^{-1/2}]\) (up to powers of \(M\)). Explicitly, the reachable dimensions are:
\[
[m^a \hbar^b] = M^{a+b}\,L^{2b}\,T^{-b}, \qquad L^{2b} = (T^{-1})^b \cdot \text{(M-factor)}.
\]

The unique time-like combination is \([\hbar/m] = L^2 T^{-1}\) (diffusion coefficient), not \(T\).

## 3. Compton Time Caveat

### 3a. Compton time formula

The Compton time is:
\[
\tau_C = \frac{\hbar}{mc^2}.
\]

Dimension check:
\[
[\hbar/(mc^2)] = \frac{ML^2T^{-1}}{M \cdot L^2T^{-2}} = T. \quad\checkmark
\]

### 3b. Requires speed of light \(c\)

\(\tau_C\) uses \(c\) (dimension \(LT^{-1}\)), which is **not** in the non-relativistic constant set \(\{m, \hbar\}\) of P4.2.

Adding \(c\) to the set expands the dimensional span:
\[
\mathrm{span}\{[m], [\hbar], [c]\} = \{M, L, T\} \quad\text{(full 3D span)}.
\]

With \(\{m, \hbar, c\}\), the matching equations for \([\varepsilon] = T\) become:
\[
[m]^a\,[\hbar]^b\,[c]^d = M^{a+b}\,L^{2b+d}\,T^{-b-d}
\]
\[
\begin{cases}
M: & a + b = 0 \\
L: & 2b + d = 0 \\
T: & -b - d = 1
\end{cases}
\implies b = -1,\; d = 2,\; a = 1.
\]

Solution: \(\varepsilon = m\hbar^{-1}c^2 \cdot (\text{dimensionless}) = \hbar/(mc^2) \cdot (\text{dimensionless})\). This is \(\tau_C\).

### 3c. P4.2 scope

P4.2 operates in non-relativistic quantum mechanics. The hypothesis set is \(\{m, \hbar\}\) without \(c\). The Compton time is **outside scope**: it belongs to relativistic QM / QFT.

**Compton time is not a counterexample to the obstruction within P4.2's framework.**

## 4. State-Dependent Scales

### 4a. de Broglie time

\[
\tau_{dB} = \frac{\hbar}{E}, \qquad [\hbar/E] = \frac{ML^2T^{-1}}{ML^2T^{-2}} = T. \quad\checkmark
\]

### 4b. Obstruction: universality

The energy \(E\) is **state-dependent** (eigenvalue of Hamiltonian, kinetic energy of a wave packet, etc.). A discrete time step \(\varepsilon\) must be a **universal lattice constant**, independent of quantum state.

If \(\varepsilon = \varepsilon(E)\), then different states evolve on different lattices. The composition law \(K_{n\varepsilon} = (K_\varepsilon)^n\) requires a single, fixed \(\varepsilon\).

**State-dependent time scales cannot serve as universal lattice spacing.**

## 5. Scale-Invariance Obstruction

### 5a. Continuum semigroup

In the continuum, composition holds for **arbitrary** \(t_1, t_2 > 0\):
\[
K_{t_1+t_2} = K_{t_1} \circ K_{t_2}, \qquad \forall\, t_1, t_2 > 0.
\]

The semigroup parameter \(t \in (0, \infty)\) carries no preferred scale.

### 5b. Discrete composition

On a time lattice with spacing \(\varepsilon\):
\[
K_{n\varepsilon} = (K_\varepsilon)^n, \qquad n \in \mathbb{N}.
\]

Composition holds only for \(t = n\varepsilon\), \(n \in \mathbb{N}\). The semigroup \((0,\infty)\) is replaced by the discrete monoid \(\varepsilon\mathbb{N}\).

### 5c. Preferred scale breaks semigroup freedom

If \(\varepsilon\) is fixed (from any source), the theory acquires a **preferred time scale**:
- Composition is restricted to multiples of \(\varepsilon\).
- The continuous semigroup \(\{K_t\}_{t>0}\) is replaced by a discrete semigroup \(\{K_\varepsilon^n\}_{n \in \mathbb{N}}\).
- The infinitesimal generator \(H = -i\hbar\,\lim_{t\to 0^+}(K_t - \mathbf{1})/t\) is replaced by a finite-difference operator.

**Tension with (C)+(D):** Hypothesis (D) forbids external parameters. If \(\varepsilon\) cannot come from \(\{m, \hbar\}\) (Section 2), it must be an external input. But external inputs violate (D). Therefore:

\[
\boxed{
\varepsilon \notin \mathrm{Image}([m]^a[\hbar]^b) \;\wedge\;
\text{(D) forbids external inputs}
\;\Longrightarrow\;
\text{discrete time violates (D).}
}
\]

## 6. Identity Limit Obstruction (Secondary)

### 6a. Finite-\(\varepsilon\) kernel

On a spatial lattice with spacing \(\ell\), the kernel \(K_\varepsilon(x_i, x_j)\) is a matrix indexed by lattice sites. For finite \(\varepsilon > 0\):
\[
K_\varepsilon(x_i, x_j) \sim \left(\frac{m}{2\pi\hbar\varepsilon}\right)^{d/2}\exp\!\left(\frac{im(x_i - x_j)^2}{2\hbar\varepsilon}\right).
\]

### 6b. Delta concentration requires \(\varepsilon \to 0\)

The identity limit (I) demands:
\[
\lim_{\varepsilon \to 0^+} K_\varepsilon(x, y) = \delta^{(d)}(x - y).
\]

For fixed \(\varepsilon > 0\), the kernel has finite width \(\sim \sqrt{\hbar\varepsilon/m}\) and cannot concentrate to \(\delta\). The identity limit is **only achievable in the continuum limit** \(\varepsilon \to 0\).

### 6c. Lattice-only formulation

If \(\varepsilon\) is strictly positive (discrete, no continuum limit), hypothesis (I) fails: there is no time value below \(\varepsilon\) at which the kernel approaches the identity.

The identity limit (I) implicitly requires the time parameter to be **dense** in a neighborhood of 0, which forces a continuum.

## 7. Combined Argument

### 7a. Logical chain

\[
\text{(D)} \;\Rightarrow\; \varepsilon \in \mathrm{Image}([m]^a[\hbar]^b)
\;\xRightarrow{\text{Section 2}}\; \text{contradiction (no solution)}.
\]

Therefore:
\[
\text{(D)} \;\Rightarrow\; \varepsilon \text{ cannot exist as internal parameter}
\;\Rightarrow\; \text{discrete time requires external input}
\;\Rightarrow\; \text{violates (D)}.
\]

### 7b. Supporting obstructions

Even if \(\varepsilon\) could somehow be fixed:
- **(Scale-invariance, Section 5):** Composition degrades from continuous semigroup to discrete monoid.
- **(Identity limit, Section 6):** Hypothesis (I) fails for fixed \(\varepsilon > 0\).

### 7c. Escape routes (and why they fail)

| Escape | Status | Reason |
|---|---|---|
| Compton time \(\tau_C = \hbar/(mc^2)\) | Blocked | Requires \(c\), not in \(\{m, \hbar\}\) |
| de Broglie time \(\tau_{dB} = \hbar/E\) | Blocked | State-dependent, not universal |
| External parameter | Blocked | Violates (D) |
| Dimensionless \(\varepsilon\) | Meaningless | \([\varepsilon] = T \neq 1\) |

## 8. Verdict

**Verdict 1: Dimensional obstruction.**

The lattice spacing \(\varepsilon\) has dimension \(T\). The non-relativistic constant set \(\{m, \hbar\}\) spans only \(\{M^\alpha L^\beta T^\gamma : \beta = -2\gamma\}\), which does not contain \(T = M^0 L^0 T^1\).

No function \(\varepsilon = f(m, \hbar)\) exists. A discrete-time formulation requires external input (either \(c\) or an independently specified scale), violating hypothesis (D).

**Continuum time is forced by dimensional homogeneity (D).**

Supporting obstructions:
- Scale-invariance (Section 5): fixing \(\varepsilon\) breaks continuous semigroup.
- Identity limit (Section 6): fixed \(\varepsilon > 0\) prevents \(K_\varepsilon \to \delta\).
