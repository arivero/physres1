# d=4 Contact Interaction Spectrum: APS Exact Solution

## 1. Setup

**Operator:** -Δ_b on ℝ⁴\{0} in L²_b = L²(ℝ⁴\{0}, r^{-4}d⁴x). s-wave sector.

**b-variable:** t = log r. L²_b = L²(ℝ, dt) (flat measure in t).

**Bound state equation:** (-Δ_b - E)ψ = 0, E < 0. Write E = -ℏ²κ²/(2m), κ > 0.

**ODE:** [d²/dt² + 2d/dt - κ²] f(t) = 0   (d=4 s-wave b-Laplacian)

## 2. Solution Space

Characteristic roots of λ² + 2λ - κ² = 0:

λ_± = -1 ± √(1 + κ²)

Since κ > 0: √(1+κ²) > 1, so:
- λ_+ = -1 + √(1+κ²) > 0   (solution decays at t→-∞, i.e. r→0)
- λ_- = -1 - √(1+κ²) < -2  (solution decays at t→+∞, i.e. r→∞)

**Behavior:**
- f_+(t) = e^{λ_+t}: L²_b on (−∞, t_0) but DIVERGES as t→+∞
- f_-(t) = e^{λ_-t}: L²_b on (t_0, +∞) but DIVERGES as t→−∞

No globally L²_b solution without boundary condition at t = −∞ (r = 0 face).

## 3. APS Boundary Condition at the Face (t → −∞)

**APS BC:** At the blown-up face S³ (= {r = 0} after blow-up = {t → −∞}), the
APS boundary condition specifies the L²_b solution from the left half-line t ∈ (−∞, t_0]:

f(t) ~ c · e^{λ_+t}   as t → −∞   [decay into the face]

**Self-adjoint extension parameter μ_APS:** The APS boundary condition at finite t = log(L₀)
(L₀ = physical UV scale, e.g. L_P or the contact range) requires:

f(t) satisfies: f(log L₀) = 0   [Dirichlet-type APS condition at t₀ = log L₀]

OR (more general): [df/dt + μ_APS · f]|_{t=t₀} = 0   [Robin-type APS condition]

The Robin parameter μ_APS ∈ ℝ is the one-parameter family (n_±=1 from B-DEF-P1.1).

## 4. Exact Spectrum from Robin APS Condition

**Solution satisfying BC at t₀ = log L₀ and decaying at t→+∞:**

General solution: f(t) = A·e^{λ_+t} + B·e^{λ_-t}

Decay at t→+∞: only B·e^{λ_-t} survives (λ_- < 0), so A = 0 for t > t₀.
Decay at t→-∞: only A·e^{λ_+t} survives (λ_+ > 0), so B = 0 for t < t₀.

**Matching condition at t = t₀ (gluing left and right solutions):**

[f]_{t₀} continuous: A·e^{λ_+t₀} = B·e^{λ_-t₀}   → A/B = e^{(λ_--λ_+)t₀}

Robin APS condition applied at t₀ from the LEFT (incoming from t<t₀):
df/dt + μ_APS · f = 0   at t = t₀
(λ_+ + μ_APS) A e^{λ_+t₀} = 0

Non-trivial solution requires: λ_+ + μ_APS = 0   → λ_+(κ) = −μ_APS

**Eigenvalue equation:**
-1 + √(1 + κ²) = −μ_APS
√(1 + κ²) = 1 − μ_APS
1 + κ² = (1 − μ_APS)²

**Bound state energy:**
κ² = (1 − μ_APS)² − 1 = μ_APS² − 2μ_APS = μ_APS(μ_APS − 2)

**Pre-squaring constraint:** √(1+κ²) ≥ 1 > 0 requires 1 − μ_APS > 0, i.e. μ_APS < 1.

**Existence condition:** κ² > 0 AND μ_APS < 1:
κ² = μ_APS(μ_APS − 2) > 0 AND μ_APS < 1 → μ_APS < 0 ONLY.
(The μ_APS > 2 branch is spurious — it fails the pre-squaring constraint. Critic C2 fix.)

**Bound state energy (in b-units κ = 1/L₀ is natural scale):**

E_B = −ℏ²κ²/(2m) = −(ℏ²/2m) · μ_APS(μ_APS − 2)

Or equivalently: E_B = −(ℏ²/2mL₀²) · μ_APS(μ_APS − 2)

where L₀ is the UV scale at which the APS condition is applied.
